Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • touch (file name).(file type): creates a new file.
  • git checkout (main, branch, etc): navigate between branches.
  • git add -A: add file contents to the index to prepare the content staged for the next commit.
  • git commit -m “[descriptive message]”: commit your staged content as a new commit snapshot
  • git pull origin main: it downloads new code from the branch named master on the remote named origin and integrates them into your local HEAD branch.
  • git push origin [branch name]: push changes to main branch

JavaScript

  • Variables - store data types as values
  • If statement: a way of introducing decision-making into our code.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal
  • Array: a single variable that is used to hold a group of data, [] brackets are used to group data. Each item is assigned an index according to the order in which it is stored in the array starting with "0".
  • Const: a constant variable whose value cannot be changed once it's assigned.
  • Let: allows us to reassign the value of the variable over time. The 2nd statement is the condition. The final statement is what allows the array to iterate over each item. [x++ is x+1] every time the code loops, x will increase by 1.
  • To comment out multiple lines of JavaScript code, put /* before and */ after the block of code. Commenting out code makes it inactive without deleting it.
  • Function: is simply a set of instructions that tells the computer how to perform a certain task. A function is different from a for loop or conditional statement because functions do not automatically execute when the JavaScript file (in our case, script.js) is run. Instead, you must call a function by name in order to execute it. A function's name is followed by parentheses ().
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
